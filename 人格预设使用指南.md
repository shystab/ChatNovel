# AI 人格预设功能使用指南

## 功能概述

AI 人格预设功能允许你为 AI 助手定制不同的写作风格和行为模式。通过创建和切换不同的人格预设，你可以让 AI 在不同场景下展现不同的写作特点。

## 功能特点

- ✅ **多人格管理**：创建、编辑、删除多个人格预设
- ✅ **一键切换**：快速在不同人格之间切换
- ✅ **实时生效**：在 AI 聊天中选择人格后立即生效
- ✅ **持久化存储**：所有预设保存在本地数据库中

## 使用步骤

### 1. 创建人格预设

1. 点击主界面右上角的"设置"按钮（齿轮图标）
2. 在设置页面左侧导航中点击"人格预设"
3. 点击右上角"创建新预设"按钮
4. 填写预设信息：
   - **预设名称**：例如"严肃学术风格"、"轻松诙谐风格"
   - **描述**（可选）：简短描述这个人格的特点
   - **System Prompt**：详细描述 AI 的行为规范和写作风格

#### System Prompt 示例

**严肃学术风格：**
```
你是一个严谨的学术写作助手。

写作要求：
- 使用正式、规范的书面语
- 避免口语化表达
- 注重逻辑性和严谨性
- 引用准确，表述客观
- 段落结构清晰，论述有条理
```

**轻松诙谐风格：**
```
你是一个充满活力的写作助手。

写作特点：
- 语言生动活泼
- 善用比喻和幽默
- 贴近生活，通俗易懂
- 注重可读性和趣味性
- 可以适当使用俏皮话和网络流行语
```

**古风武侠风格：**
```
你是一位精通武侠小说创作的文豪。

写作风格：
- 使用典雅的文言白话
- 善用武侠特有的描写手法
- 注重江湖氛围的营造
- 人物对话符合江湖侠客身份
- 动作描写生动传神
```

### 2. 管理人格预设

在人格预设列表中，你可以：

- **编辑**：点击预设卡片右侧的"编辑"图标（铅笔）
- **启用/禁用**：点击"星星"图标切换启用状态（同一时间只能启用一个）
- **删除**：点击"垃圾桶"图标删除预设

### 3. 在 AI 聊天中使用人格

1. 回到主编辑界面
2. 在右侧 AI 助手面板的顶部工具栏
3. 点击"用户"图标（User），打开人格选择器
4. 选择你想使用的人格预设
5. 之后的所有对话都会使用选中的人格

**提示**：
- 选择"默认人格"将使用系统内置的写作助手人格
- 当前启用的预设会显示"启用"标签
- 选中的人格会在图标上显示"✓"标记

## 技术实现

### 后端 API

人格预设功能基于以下 REST API：

- `GET /api/v1/presets/` - 获取所有预设列表
- `POST /api/v1/presets/` - 创建新预设
- `PUT /api/v1/presets/{id}` - 更新预设
- `PATCH /api/v1/presets/{id}/toggle` - 切换启用状态
- `DELETE /api/v1/presets/{id}` - 删除预设

### WebSocket 集成

在 AI 对话中，`preset_id` 参数会通过 WebSocket 传递给后端：

```typescript
{
  type: "chat",
  messages: [...],
  preset_id: 1,  // 指定使用的人格预设 ID
  max_length: 500,
  use_memory: true
}
```

后端会根据 `preset_id` 加载对应的 `system_prompt`，注入到对话上下文中。

### 数据模型

```python
class Preset(SQLModel, table=True):
    id: int | None = Field(default=None, primary_key=True)
    name: str = Field(index=True)
    description: str = Field(default="")
    system_prompt: str
    is_enabled: bool = Field(default=False)
    created_at: datetime
    updated_at: datetime
```

## 最佳实践

### 1. 人格设计原则

- **明确定位**：每个人格应该有清晰的定位和特点
- **具体指令**：System Prompt 要具体，避免模糊的要求
- **举例说明**：可以在 Prompt 中加入示例，帮助 AI 理解
- **分类管理**：根据写作场景分类创建人格（学术、小说、随笔等）

### 2. System Prompt 编写技巧

**DO（推荐）：**
- ✅ 明确列出写作要求（用条目形式）
- ✅ 说明语言风格和语气
- ✅ 指定输出格式和结构
- ✅ 提供具体的写作示例

**DON'T（避免）：**
- ❌ 模糊的要求（如"写得好一点"）
- ❌ 矛盾的指令（既要幽默又要严肃）
- ❌ 过长的 Prompt（建议控制在 500 字以内）
- ❌ 包含敏感或不当内容

### 3. 使用场景示例

| 场景 | 推荐人格 | 说明 |
|------|---------|------|
| 写论文 | 严肃学术风格 | 规范、客观、严谨 |
| 写网文 | 轻松诙谐风格 | 生动、有趣、接地气 |
| 写古风小说 | 古风武侠风格 | 典雅、意境、江湖感 |
| 写营销文案 | 营销推广风格 | 有吸引力、煽动性强 |
| 写技术文档 | 技术文档风格 | 准确、清晰、逻辑性强 |

## 故障排查

### 问题：切换人格后 AI 回复没有变化

**解决方案：**
1. 确认人格已在下拉菜单中选中（有 ✓ 标记）
2. 清空当前对话（点击"刷新"图标），重新发起对话
3. 检查 System Prompt 是否编写清晰

### 问题：创建预设时提示错误

**解决方案：**
1. 检查必填字段是否都已填写（名称和 System Prompt）
2. 检查网络连接和后端服务状态
3. 查看浏览器控制台的错误信息

### 问题：无法启用某个预设

**解决方案：**
1. 系统同时只能启用一个预设
2. 启用新预设时，其他预设会自动禁用
3. 如果启用失败，刷新页面重试

## 进阶功能

### 动态人格注入

在 WebSocket 通信中，前端可以动态指定 `preset_id`，即使该预设未"启用"也可以使用。这允许实现以下高级功能：

1. **临时人格切换**：在某次对话中临时使用特定人格，不影响全局设置
2. **多人格对话**：在同一对话中动态切换多个人格
3. **A/B 测试**：对比不同人格的输出效果

### 预设模板库（即将推出）

未来版本将支持：
- 预设导入/导出功能
- 预设模板市场
- 社区分享优秀人格预设

## 总结

人格预设功能让 AI 写作助手更加灵活和个性化。通过精心设计不同的人格，你可以让 AI 在各种写作场景下都能提供最合适的帮助。

**核心价值：** 让 AI 成为真正懂你的多面写作助手，而不是一个"千篇一律"的工具。
